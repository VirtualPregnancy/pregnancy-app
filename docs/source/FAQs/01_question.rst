Which data files are used to manage content and where are they located?
==========================================================================

.. include:: ../style.rst

The location of the data files is the ‘assets/data’ folder. These include,
    - :blue:`topics.json`, which contains all information related to the topics along with their subtopics. 
    - :blue:`videos.json`, specifies the keys and location of the videos.
    - :blue:`Markdown`, folder contains individual markdown files for each subtopic.
